# CoolRoute (prototype)

Single-page React prototype: mobile-first “climate navigation” UI with a real OpenStreetMap map (Leaflet). All weather and zone data is mocked in the frontend.

## Run locally

```bash
cd coolroute-prototype
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Edit mock data

See `src/data/mockData.ts` for username, dashboard numbers, zones, route polyline, and recommendation copy.
